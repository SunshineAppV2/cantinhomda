# Guia Rápido: Sistema Cantinho do Desbravador

Este tutorial explica como utilizar as funcionalidades de **Ranking** e **Chamada** do sistema.

---

## 1. Gestão de Frequência (Chamada)

A tela de Chamada permite registrar a participação dos membros em eventos e reuniões de forma rápida.

### Como realizar a chamada:

1.  **Identifique o Evento**: O cabeçalho da janela mostra o nome do evento (ex: "Evento: TESTE") e a data.
2.  **Lista de Membros**: Todos os desbravadores ativos são listados com suas respectivas fotos e cargos.
3.  **Critérios de Avaliação**: Para cada membro, marque as caixas de seleção correspondentes:
    *   ✅ **Presença**: O desbravador compareceu ao evento.
    *   ✅ **Pontualidade**: Chegou no horário marcado (vale pontos extras).
    *   ✅ **Uniforme Completo**: Está vestindo o uniforme correto para a ocasião (vale pontos extras).
4.  **Facilitadores**:
    *   Use o botão **"Marcar Todos"** no topo para selecionar todos os critérios de uma vez para todos os membros (útil quando a maioria cumpre os requisitos).
    *   Desmarque individualmente quem não cumpriu algum requisito.

---

## 2. Acompanhamento de Desempenho (Rankings)

O módulo de Ranking foi desenhado para incentivar a participação e o desenvolvimento dos desbravadores de forma visual e interativa.

### Ranking Geral
O painel principal exibe a classificação geral dos desbravadores.

*   **Abas de Navegação**: Utilize os botões superiores para alternar entre a visão de **Top Desbravadores** (individual) e **Top Unidades** (coletivo).
*   **Faixas Etárias**: O sistema separa automaticamente os desbravadores por faixa etária (ex: Faixa A e Faixa B) para uma comparação justa.
*   **Pontuação em Tempo Real**: Os pontos são calculados automaticamente com base na frequência, cumprimento de requisitos e atividades.

### Ranking de Unidades
Nesta visão, você acompanha a performance de cada unidade.

*   **Pódio Interativo**: As unidades com melhor desempenho (1º e 2º lugares) recebem destaque visual (cartões Ouro e Prata).
*   **Métricas de Qualidade**: Além do total de pontos, o sistema exibe a **média de pontos por membro**, garantindo que unidades menores também tenham chance de competir de igual para igual.

---

*Dica: Mantenha os registros sempre atualizados para que o Ranking reflita a realidade do clube!*
